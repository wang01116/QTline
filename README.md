## 安装方法

### Chrome / Edge

1. 下载并**解压** `QTline_v2.1.zip`
2. 打开浏览器，地址栏输入：
   - Chrome：`chrome://extensions`
   - Edge：`edge://extensions`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**，选择解压后的 `qwen-voyager` 文件夹
5. 安装完成，访问任意支持的 AI 站点即可看到右侧时间轴和右下角悬浮球

> ⚠️ **升级前建议在「设置 → 数据管理」中导出完整备份。**

## 功能介绍

### 📍 对话时间轴

右侧（默认）的纵向轴，记录当前对话的每一条用户消息：

- **悬停圆点** → 弹出消息摘要气泡（含时间、序号）
- **点击圆点** → 页面平滑滚动到该消息并高亮闪烁
- **顶部切换** → 💬 消息时间轴 / 📎 文件&图片时间轴
- **快捷键**（可自定义）：
  - `Alt+[` 跳到上一条
  - `Alt+]` 跳到下一条

时间轴位置可在设置中切换：右侧竖向 / 左侧竖向 / 顶部横向 / 底部横向。

---

### 💡 提示词库

管理常用的 AI 提示词，跨平台共用：

- **新增 / 编辑 / 删除**提示词
- **文件夹**分类（支持多层嵌套）
- **Tag** 标签筛选
- **搜索**（标题 + 内容实时搜索）
- **紧凑模式**（一行显示，更多内容）
- **可见范围**：「所有AI通用」或「仅限当前AI」
- **导出 / 导入**（JSON 格式）
- **使用方式**：点击「▶ 使用」直接填入当前对话框（可在设置中切换为仅复制）

---

### 📁 对话文件夹

将 AI 对话按主题分类归档：

- **新建文件夹**（支持多层嵌套、颜色标记）
- **添加对话**：扫描侧边栏，一键归入文件夹（支持设置 Tag）
- **共享文件夹**：可跨 AI 平台使用，保存多个平台的对话链接
- **一键展开 / 折叠**所有文件夹
- **查看所有AI的文件夹**（🗂 按钮），在一个面板内浏览所有平台的对话记录
- **点击对话链接**直接跳转至原始对话
- **导出 / 导入**（按站点独立导出）

---

### 🔗 分享到其他AI

将当前内容发送到一个或多个其他 AI：

1. **选中页面文字** → 自动弹出「🔗 分享」浮动按钮
2. 点击按钮 → 打开分享面板
3. 选择目标 AI（可多选）
4. 可选附加提示词
5. 点击发送 → 在新标签页打开目标 AI

---

### ✍️ 全屏输入

专为长文本设计的大型输入框：

- 覆盖全屏，舒适编写长段内容
- `Ctrl+Enter` 快速填入对话框
- **草稿功能**：随时保存、加载草稿（按站点分类）

---

### 🗑️ 批量删除对话

- 扫描侧边栏对话列表
- 支持全选 / 多选
- 一键批量删除

---

### 🌐 实用站点

内置常用工具快速入口（Markdown编辑器、JSON格式化、Regex测试等），支持自定义添加、导入导出。

---

### 🌙 深色模式切换

对支持深色模式的 AI 站点，一键切换深色/浅色。

---

### ⚙️ 设置

完整的侧边栏式设置面板，按功能分类：

| 分类 | 设置项 |
|------|--------|
| 外观 | 插件深色模式（跟随/始终深色/始终浅色） |
| 时间轴 | 位置、自定义颜色、防自动跳转 |
| 提示词 | 点击行为（填入/复制） |
| 草稿 | 自动保存开关 |
| 快捷键 | 自定义打开菜单、上/下节点快捷键 |
| 通用功能 | 显示对话标题、Mermaid渲染、引用回复、公式格式 |
| 数据管理 | 导出全部设置、导入设置 |
| 关于 | 版本号、GitHub链接、检查更新、更新日志 |

---

## 悬浮球操作

- **单击** → 展开功能菜单
- **拖拽** → 移动到屏幕任意位置（菜单跟随）
- 可在弹窗中关闭悬浮球显示

---

## 插件弹窗（工具栏图标）

点击浏览器工具栏图标，可以：
- 查看当前激活的 AI 站点
- 开关站点启用、悬浮球
- 快捷打开任意功能面板

---

## 键盘快捷键（默认）

| 快捷键 | 功能 |
|--------|------|
| `Alt+Q` | 打开/关闭菜单 |
| `Alt+[` | 时间轴上一条 |
| `Alt+]` | 时间轴下一条 |

> 可在「设置 → 快捷键」中自定义。

---

## 数据说明

- 所有数据（提示词、文件夹、设置）均存储在浏览器本地 `chrome.storage.local`
- 不上传任何服务器，完全离线运行
- 建议定期在「设置 → 数据管理 → 导出所有设置」备份
