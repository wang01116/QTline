## 🚀 简介

**QTline** 是一款专为 各类打造的浏览器增强插件，灵感来源于 Voyager 的导航理念。它在不改变AI对话工具核心体验的前提下，新增对话时间轴、提示词管理、批量删除和文件夹、草稿、联动不同ai等实用功能，大幅提升日常使用效率。

---

## ✨ 功能一览

### 1. 📍 对话时间轴

页面右侧出现一条纵向时间轴，每条用户消息对应一个圆点，均匀分布于轴上。

| 操作 | 效果 |
|------|------|
| 鼠标悬停圆点 | 弹出气泡：显示消息时间 & 缩略内容 |
| 点击圆点 | 页面平滑滚动至该消息并高亮闪烁 |
| 点击上方图标 | 切换「对话模式」与「文件模式」 |

- 所有消息**均匀**分布在时间轴上，无论消息数量多少都不会重叠
- 支持跨日期对话——非当天消息显示完整日期（如 `05/08 14:30`）
- 新增消息后自动更新圆点

---

### 2. 💡 提示词库

快速管理和使用常用提示词，支持标签分类。

**主要功能：**
- ➕ 新增提示词（标题 + 内容 + 多个 Tag）
- ✏️ 编辑 / 🗑️ 删除已有提示词
- 🏷️ 按 Tag 筛选提示词
- ▶️ 一键填入 Qwen 输入框（无法填入时自动复制到剪贴板）
- 📤 导出为 JSON 文件
- 📥 从 JSON 文件导入（追加模式，不覆盖已有数据）

---

### 3. 🗑️ 批量删除对话

告别逐条删除，一次处理多个对话记录。

1. 点击悬浮球 → 「批量删除」
2. 点击「扫描对话列表」——插件扫描左侧对话历史
3. 勾选想要删除的对话（支持全选）
4. 点击「删除选中」确认

> ⚠️ 此操作不可撤销，请谨慎选择。

---

### 4. 📁 文件夹功能

五种主题一键切换，偏好持久保存：

| 主题 | 说明 |
|------|------|
| 默认 | Qwen 原始样式 |
| 暗色 | 深色背景，护目夜间模式 |
| 护眼 | 浅绿色背景，降低蓝光刺激 |
| 阅读 | 暖黄纸张色，适合长时间阅读 |
| 午夜 | 极深黑背景，GitHub 暗色风格 |

主题样式通过注入 CSS 实现，QTline 自身 UI **不受任何主题影响**，始终保持完整显示。

---

### 5. 🔄 重置页面样式

一键清除所有主题注入，恢复 Qwen 默认页面样式。可通过：
- 悬浮球菜单 → 「重置页面样式」
- 插件弹窗 → 「重置页面样式」按钮

---

### 6. 🎯 可拖动悬浮球

右下角悬浮球支持自由拖拽，放到屏幕任意位置。单击弹出功能菜单，拖拽时不触发菜单。

## 📦 安装方法

### Chrome

1. 下载并解压 `qtline.zip`
2. 打开 `chrome://extensions`
3. 右上角开启 **开发者模式**
4. 点击 **「加载已解压的扩展程序」**
5. 选择解压后的 `qwen-voyager` 文件夹

### Edge

1. 下载并解压 `qtline.zip`
2. 打开 `edge://extensions`
3. 左下角开启 **开发者模式**
4. 点击 **「加载解压缩的扩展」**
5. 选择解压后的 `qwen-voyager` 文件夹

## ⚙️ 数据存储

所有数据（提示词、主题偏好、功能开关状态）均存储在浏览器本地（`chrome.storage.local`），**不上传任何服务器**，完全离线运行。

## 🔧 常见问题

**Q: 时间轴没有圆点？**  
A: 需要先在当前对话页发送消息，或等待页面完全加载后刷新一次。

**Q: 提示词无法自动填入？**  
A: Qwen 输入框结构变化时会自动改为复制到剪贴板，粘贴（Ctrl+V）即可。

**Q: 切换主题后部分样式还是有问题？**  
A: 点击「重置页面样式」后再重新选择主题，或刷新页面重试。

**Q: 批量删除没有效果？**  
A: 插件会尝试触发 Qwen 内置删除按钮；若页面结构更新导致无法识别，需手动操作。

**Q: 怎么隐藏悬浮球？**  
A: 点击浏览器工具栏图标 → 关闭「悬浮球」开关；或在悬浮球菜单中也可操作（重新打开需通过工具栏弹窗）。
